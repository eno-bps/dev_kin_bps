# dev_kin_bps

BPS社内kintone（bps.cybozu.com）のカスタマイズファイル管理リポジトリ。

## 構成

```
dev_kin_bps/
├── kintone.config.yaml    # kintone MCP デフォルトドメイン設定
└── 57_vacation_calendar/  # 休暇申請アプリ カスタムカレンダー
    ├── vacation-calendar.js
    └── vacation-calendar.css
```

## アプリ別カスタマイズ

### 57: 休暇申請 - カスタムカレンダー

カレンダーPlusプラグインの代替として作成したタイムラインカレンダー。

#### 機能

- 月次タイムライン表示（縦：申請者、横：日付）
- スペース「社内届け出書類」（ID:4）のメンバー全員を表示
- 休暇種別ごとの色分け（有給休暇、特別休暇、シーズン休暇、振替休日、無給休暇、遅刻/早退、忌引き、その他）
- ステータス別スタイル（承認済み＝塗りつぶし、申請中＝半透明+斜線）
- 時間休のマウスオーバーで時刻表示
- 連続休暇のバー表示（角丸で開始/終了を表現）
- 土日・当日ハイライト
- イベントクリックでレコード詳細を別タブ表示
- 一覧へ戻るボタン

#### kintone設定手順

1. **カスタマイズビュー作成**
   - アプリの設定 → 一覧 → 新規作成
   - 表示形式：カスタマイズ
   - HTML：`<div id="vc-root"></div>`

2. **JS/CSSアップロード**
   - アプリの設定 → JavaScript/CSSでカスタマイズ
   - PC用JavaScript：`vacation-calendar.js`
   - PC用CSS：`vacation-calendar.css`

3. **カレンダーPlusプラグイン無効化**（任意）
   - プラグイン設定から無効にする
